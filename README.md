# SQL-intermedio
SQL a nivel intermedio utilizando la DB Northwind
**flujo de aprendizaje lógico** (de lo estructural a lo analítico):

### 1. Cimientos y Estructura (Nivel Esencial)

* **Data Types & Primary vs. Foreign Keys:** La base de cualquier tabla.
* **Aliasing:** Legibilidad desde el primer día.
* **Joins & Unions:** Cómo conectar y apilar datos.

### 2. Manipulación y Lógica (Nivel Intermedio I)

* **Updating / Deleting:** Modificación de registros existentes.
* **Having vs. Group By:** Filtrado de agregaciones.
* **Case:** Lógica de "si/entonces" en tus reportes.
* **GetDate & String Functions:** Limpieza de fechas y textos (Trim, Replace, Upper, etc.).

### 3. Análisis Avanzado (Nivel Intermedio II)

* **Partition By (Window Functions):** Cálculos complejos por grupos.
* **CTEs & Subquerys:** Organización de consultas anidadas.
* **Creating Views:** Persistencia de consultas frecuentes.

### 4. Automatización y Admin (Nivel Avanzado)

* **Temp Tables:** Gestión de datos volátiles en memoria.
* **Store Procedures:** Automatización de procesos.
* **Import/Export Data:** Movimiento de información entre fuentes (Excel, CSV, etc.).
* **Sys Tables & Regular Expressions:** Auditoría del sistema y búsqueda de patrones complejos.
